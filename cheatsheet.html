This is my Cheatsheet for Position and Display Attributes

Position Attributes

Attribute Description
Position Allows us to change the positioning of the element we choose, even allowing us to remove the element from the normal flow
Static The default position of elements, it positions the element under the element above
relative Allows us to offset an element a specific distance from it's original position
Absolute Allows us to remove the element from the pages original flow, allowing us to place the element in an absolute position on the page
Fixed Allows us to fix an element to the page, which means the element won't scroll with the page
Sticky Allows us to fix an element to the page up to a specified point where the element will rejoin the original flow

Display Attributes

Attirbute Description
Inline Gives the element a tightly wrapped border and other elements can be place on the left and right, can't adjust width or height
Block Allows changing width and height but other elements cannot be placed on the left or right, takes up whole horizontal page
Inline-Block Takes properties from both inline and block, can change the width and height and other elements can be placed on the sides.
